# Minima OS

An AI-first Android launcher. No app grid. No home screen. Just a clock, a command bar, and a memory.

You tell Minima what you want in natural language — "text Sarah I'm running late", "remind me to call mom at 6", "what did I eat yesterday", "good morning" — and it figures out the intent, executes the task, and remembers anything worth remembering.

## Features

- **GPT-4o intent classification** with deterministic fallback and an `ANSWER` catch-all (nothing ever returns "didn't understand")
- **3-tier memory** — STM (48h) → MTM (30d) → LTM (permanent), auto-promotion based on access patterns
- **Personal knowledge graph** — people, places, patterns, preferences extracted from completed tasks
- **AI-driven context surface** — insight cards generated by GPT-4o from your memory + time context
- **AI-driven proactive layer** — morning briefs, nudges, people reminders; learns from dismissals
- **16 intents** — open app, call, text, email, calendar, alarm, reminder, DnD, navigate, search, translate, math, joke, remember, recall, answer
- **Fuzzy app matching** — Levenshtein edit distance, handles typos and misspellings
- **3 sensitivity modes** — Quiet (1 card), Normal (3), Proactive (5)

## Architecture

7 Gradle modules, Hilt DI throughout:

```
app         → Application, MainActivity, Hilt entry point
ui          → Jetpack Compose screens (launcher, memory, settings)
agent       → TaskExecutor, IntentClassifier, orchestration
capability  → Pluggable action handlers (chat, memory, system, messaging, ...)
data        → Room database, MemoryManager, ContextEngine, ProactiveEngine
model       → CloudModelProvider (GPT-4o), LocalModelProvider (future)
core        → Shared models (Intent, Task, StepResult)
```

See `ARCHITECTURE.md` for the full diagram.

## Build

```bash
export JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home"
export ANDROID_HOME="$HOME/Library/Android/sdk"
./gradlew :app:assembleDebug
```

APK output: `app/build/outputs/apk/debug/app-debug.apk`

## Install

```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
adb shell monkey -p com.minima.os -c android.intent.category.LAUNCHER 1
```

Press Home and select Minima as the default launcher.

## API Key Setup

Minima needs an OpenAI API key to use GPT-4o. Either:

1. **In-app**: open Settings → paste your key
2. **Via adb** (debug builds only):
   ```bash
   adb shell "run-as com.minima.os sh -c 'cat > shared_prefs/minima_prefs.xml'" <<EOF
   <?xml version="1.0" encoding="utf-8" standalone="yes" ?>
   <map>
       <string name="openai_api_key">sk-...</string>
   </map>
   EOF
   ```
